# 🏥 Hospital Resource Optimization System

## 📌 Project Overview
This project focuses on optimizing hospital resources such as doctors, appointments, and patient management using MySQL and Power BI.

## 🛠 Tools Used
- MySQL
- Power BI
- Excel
- SQL
- ER Diagram

## 📊 Key Features
- Doctor availability tracking
- Appointment scheduling analysis
- Revenue calculation
- Resource utilization KPIs
- Dashboard visualization

## 📈 KPIs Created
- Total Patients
- Total Appointments
- Revenue Generated
- Average Consultation Fee
- Doctor Utilization Rate

## 🗂 Database Structure
Tables:
- Doctors
- Patients
- Appointments
- Departments
)

## 🚀 Conclusion
This system helps hospitals efficiently manage resources and improve operational performance.
